# FamilyParametersOrder

Плагин FamilyParametersOrder, адаптирован из Revit.SDK, который демонстрирует сортировку параметров семейств в алфавитном порядке английского алфавита (прямом и обратном порядке).

## Описание

Данный плагин реализует два способа сортировки параметров семейств:

1. Сортировка параметров в семействах, сохранённых в файлах на диске (.rfa).
2. Сортировка параметров в семействах, которые уже загружены в проект Revit.

## Категория

Семейства, Параметры

## Требования

Autodesk Revit 2022

## Установка

1. Скопируйте файлы addin-файл в папку  
   `C:\ProgramData\Autodesk\Revit\Addins\2022\`
2. Скопируйте файлы плагина (DLL, изображения) в папку  
   `C:\ProgramData\Autodesk\Revit\Addins\2022\FamilyParametersOrder`
3. Запустите или перезапустите Revit 2022.
4. Плагин появится на панели "**Sort parameters**" .

## Использование

- Для сортировки параметров семейства из файла вызовите соответствующую команду плагина и укажите путь к файлу семейства.
- Для сортировки параметров уже загруженных в проект семейств выберите нужное семейство в проекте и примените сортировку.
